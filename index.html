This project provides a site to display the source data and scatter plots to compare lattitude to various features of weather such as max temperature, wind speed, cloudiness and humidity. To create these plots, data on 500 cities was pulled from the OpenWeatherMaps API into a jupyter notebook where relevant data was sorted into lists and then made into a dataframe through Pandas. Using the data from the newly made dataframe, plots were created with Matplotlib in order to observe trends between latitude and aspects of weather, which will be explored on this site.
